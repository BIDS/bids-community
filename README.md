## Summer Book Club 2026

This year the BIDS staff are reading [Unreasonable Hospitality](https://www.unreasonablehospitality.com/books) by Will Guidara.

Kirstie recommended it saying:

> I think there are a lot of points we can take around running events that are of the highest possible standards, where we welcome and inspire the participants, and end up with (some of) them converting to active contributors in a global open source community.
> 
> As we start to build a bigger team here at BIDS I'm really keen to deepen our shared understanding of the culture we want to collectively create.

Everyone is invited to add their comments to the corresponding [`summer-book-club`](https://github.com/BIDS/bids-community/issues?q=state%3Aopen%20label%3A%22summer-book-club%22) issue before Thursday at noon following this schedule:
